# joop-base [![Build Status][travis.svg]][travis] [![Docker Build][docker.svg]][docker]

A Docker base image for [`naftulikay/joop`][joop], providing a Jupyter execution environment with support for:

 - Python 3.7
 - [Rust][rust] (via [`evcxr`][evxcr])
